# animation-manifest.md
## Motion Ruleset

---

# 1. PAGE LOAD ANIMATION

Allowed:
- Opacity fade-in
- translateY (10–20px)
- Staggered delay 80–120ms

Duration:
< 800ms total

---

# 2. HOVER ANIMATION

Allowed:
- Border color transition
- Glow increase
- Slight scale (max 1.02)

Not allowed:
- Bounce
- Spin
- Elastic motion
- Infinite floating

---

# 3. STATUS INDICATOR

Pulse:
- Scale subtle
- 2–3s cycle
- No harsh glow spikes

---

# 4. TRANSITIONS BETWEEN PAGES

If implemented:
- Must be subtle fade
- Not dramatic slide
- No cinematic effects

---

# 5. MOTION PHILOSOPHY

Motion should feel:
System-level.
Intentional.
Mechanical.

Not:
Playful.
Cartoonish.
Over-produced.

---

END
